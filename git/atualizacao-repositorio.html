Passos para atualização do repositório original

  1. Crie a sua conta no Github.
  2. Realize um fork do repositório principal, em https://github.com/denilsonfag/programacao-internet, o que vai criar uma cópia do repositório na sua conta.
  3. Utilizando um ambiente de linha de comando, escolha um local no sistema de arquivos onde vai baixar uma cópia do repositório na sua máquina
  4. Execute a cópia com o comando:

    git clone https://github.com/nome-da-sua-conta/programacao-internet.git

  5. Caso você já tenha realizado os passos acima anteriormente, é necessário atualizar o seu repositório com as alterações do repositório principal. Para isso:
    1. Configure o repositório principal como um dos remotos do seu repositório local. Para isso, execute:

      git remote add upstream https://github.com/denilsonfag/programacao-internet.git

    2. Baixe as alterações: git fetch upstream
    3. Mescle as alterações com o seu repositório local: git merge upstream/master
    4. Atualize o repositório na sua conta do Github: git push
  6. Realize as alterações necessárias e realize seus commits. Ver Funcionamento básico.
  7. Atualize o repositório na sua conta do Github: git push
  8. Na sua conta do Github, solicite o pull request
