Propriedade display

Todos os elementos HTML tem um valor padrão para a propriedade display, que pode ser alterado. inline e block são os valores mais comuns.

Elementos inline não podem conter elementos block. Elementos block podem conter elementos block e/ou inline.

none

Um elemento com display:none não será exibido e não terá seu espaço reservado, como se não existisse.

inline

Elementos com display:inline ocupam apenas o espaço necessário ao seu conteúdo, permitindo que outros elementos sejam exibidos ao seu lado. Não permitem a definição da largura e da altura. Exemplo do posicionamento de elementos span, que são inline por padrão (redimensione a janela para verificar como os elementos se ajustam):

span1 - Lorem ipsum dolor sit amet. span2 - Lorem ipsum dolor sit amet. span3 - Lorem ipsum dolor sit amet. span4 - Lorem ipsum dolor sit amet.

block

Elementos com display:block ocupam toda a linha em que estão inseridos, não permitindo a exibição de outros elementos ao seu lado. Permitem a definição da largura e da altura do bloco. Exemplo do posicionamento de elementos div, que são block por padrão (redimensione a janela para verificar como o texto de ajusta dentro dos blocos):

div1 - Lorem ipsum dolor sit amet, consectetur adipiscing elit.
div2 - Lorem ipsum dolor sit amet, consectetur adipiscing elit.

inline-block

Possui características dos dois métodos de exibição anteriores. Devido à forma como os blocos se ajustam, pode ser utilizado para criação de layouts responsivos. Os elementos div abaixo foram configurados com display:inline-block:

div1 - Lorem ipsum dolor sit amet.
div2 - Lorem ipsum dolor sit amet.
div3 - Lorem ipsum dolor sit amet.
div4 - Lorem ipsum dolor sit amet.
